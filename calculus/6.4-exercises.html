5.5 Exercises

Note: A calculator is required for these problems.

1. Let R represent the region bounded by \(y = \sqrt{x}\), the \(x\)-axis, and the line \(x = 4\). Find the volume of the solid generated by rotating R about the \(x\)-axis.

2. Let P represent the region bounded by \(f(x) = x^2 + 3\), the \(y\)-axis, and the line \(y = 12\). Find the volume of the solid generated when P is rotated about the \(y\)-axis.

3. Find the volume of the solid generated by rotating the region bounded by the function \(g(x) = x - 4\), the \(y\)-axis, and the line \(x = 2\) about the \(x\)-axis.

4. The region A is bounded above by the line \(y = 4\) and below by the function \(y = x^2\). Find the volume of the solid generated when A is rotated about the line \(y = 4\).

5. Find the volume generated when one of the waves of the function \(y = 2\cos{\frac{x}{5}}\) is rotated about the \(x\)-axis.

6. Let B denote the region enclosed by the \(y\)-axis, the function \(y = 5 - x\), and the function \(y=e^x\). Find the volume of the solid generated when B is rotated about the line \(x = 0\).

7. Find the volume of the solid generated when the region bounded by the \(x\)-axis and the lines \(y=2x\) and \(y=3-x\) is rotated about the line \(y = 0\).

8. The area bounded by the function \(y = 4 -x^2\), the \(y\)-axis, and the line \(x = 1\) will be rotated about the line \(y = 3\). Find the volume of the solid.

9. Let R be the region bounded by the \(x\)-axis, the \(y\)-axis, and the function \(y=e^{-2x}\). Find the volume of the solid generated by rotating R about the \(x\)-axis or show that it diverges.

10. Let P be the region bounded by the line \(x = 2\), the line \(y=1\), and the function \(y=1 -\frac{1}{x-1}\). Find the volume generated when P is rotated about the line \(y=1\) or show that it diverges.

11. Show that the volume of the solid generated by rotating the region bounded by the line \(y=0\) and the function \(y=\sqrt{\ln{x}}\) about the \(x\)-axis diverges.

12. The area bounded by the \(x\)-axis, the line \(x = 1\), the line \(x = k\), and the function \(y = \sqrt{3x}\) is rotated about the \(x\)-axis. Find the value of \(k\), \(k \gt 1\), for which the volume of the generated solid is 69 cubic units.

13. Let R denote the area bounded above by the \(x\)-axis and below by the function \(f(x) = x^2 - kx + 5\). Find the value of \(k\) for which the solid formed by rotating R about the line \(y = 0\) has a volume of 0.69 cubic unit.
