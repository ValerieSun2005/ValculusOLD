0.2 Exercises — Problem 6

(6) Let \(f(x) = 2x^2 + 5x + c\). Find the value of \(c\) for which \(f\) has one real zero.

Let \(\Delta = b^2 - 4ac\). We know that if \(\Delta = 0\), then \(f(x) = 0\) has only one real solution. In this case, $$ \Delta = (5)^2 - 4(2)c = 25 - 8c. $$ Setting \(\Delta\) equal to \(0\), we obtain $$ 25 - 8c = 0 \implies c = \bbox[border: 2px solid white, 2pt]{\frac{25}{8}.} $$
