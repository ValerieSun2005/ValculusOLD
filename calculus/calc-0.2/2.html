0.2 Exercises — Problem 2

(2) Find the values of \(x\) for which \(3x^2 + 7x = 13\).

For this problem, let us rewrite the equation in the form \(\displaystyle ax^2 + bx + c = 0\): $$ 3x^2 + 7x - 13 = 0, $$ for which we will use the quadratic formula to find the zeros of \(x\). $$ 3x^2 + 7x - 13 = 0 \implies a = 3, b = 7, c = -13.$$ $$ \therefore x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} = \frac{-(7) \pm \sqrt{(7)^2 - 4(3)(-13)}}{2(3)} \implies $$ $$ x = \frac{-7 \pm \sqrt{205}}{6}.$$ $$ x = \bbox[border: 2px solid white, 2pt]{\frac{-7 + \sqrt{205}}{6}, \frac{-7 - \sqrt{205}}{6}.} $$
