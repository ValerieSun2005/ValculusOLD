0.2 Exercises — Problem 9

(9) Find the minimum point of the function \(h(x) = x^2 + 10x - 4.\)

\(h(x)\) is a parabola that opens upward; therefore, we seek to find the vertex, which represents—in this case—the minimum point. We can either do so by completing the square or by using the property that \(\displaystyle x = -\frac{b}{2a}\) for the vertex. $$ x = -\frac{b}{2a} = -\frac{10}{2(1)} = -5. $$ We will now plug \(x = -5\) in to find \(h(-5)\), the \(y\)-coordinate of the vertex. $$ h(-5) = (-5)^2 + 10(-5) -4 = -29. $$ The minimum point is therefore \(\bbox[border: 2px solid white, 2pt]{(-5, -29).}\)
