0.2 Exercises — Problem 4

(4) How many real zeros does the function \(4x^2 + 3x + 12\) have?

Let \(\Delta = b^2 - 4ac\). Recall, from the section notes, that if \(\Delta > 0,\) then the quadratic has two real solutions. If \(\Delta = 0\), then the quadratic has only one real solution. If \(\Delta < 0\), then the quadratic has no real solutions. In this case, $$ a = 4, b = 3, c = 12 \implies $$ $$ \Delta = b^2 - 4ac = (3)^2 - 4(4)(12) = -183. $$ Because \( \displaystyle \Delta < 0\), the function has no real solutions.
