0.2 Exercises — Problem 3

(3) Find the values of \(x\) for which \(-5x^2 + 12x = 2x - 4\).

For this problem, let us rewrite the equation in the form \(\displaystyle ax^2 + bx + c = 0\): $$ -5x^2 + 12x = 2x - 4 \implies -5x^2 + 10x + 4 = 0,$$ for which we will use the quadratic formula to find the zeros of \(x\). $$ a = -5, b = 10, c = 4.$$ $$ \therefore x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} = \frac{-(10) \pm \sqrt{(10)^2 - 4(-5)(4)}}{2(-5)} \implies $$ $$ x = \frac{10 \pm \sqrt{180}}{10} = 1 \pm \frac{3 \sqrt{5}}{5}.$$ $$ x = \bbox[border: 2px solid white, 2pt]{\frac{1 + 3\sqrt{5}}{5}, \frac{1 - 3\sqrt{5}}{5}.} $$
