\(28.\)

The function \(g\) has derivatives of all orders. It is known that \(g(2) = -3, g(3) = 0,\) and \(\displaystyle \int_2^3 g(x) \, \dd x = 8.\) Find the value of \(\displaystyle \int_2^3 x g'(x) \, \dd x.\)


\(29.\)

The functions \(f\) and \(g\) have derivatives of all orders. It is known that \(f(2) = 0, f(3) = -2, g(3) = 5,\) and \(\displaystyle \int_2^3 f(x) g'(x) \, \dd x = 13.\) Find the value of \(\displaystyle \int_2^3 f'(x) g(x) \, \dd x.\)


\(30.\)

Let \(p\) be a differentiable function. Show that \(\displaystyle \int p(x) \, \dd x = x p(x) - \int x p'(x) \, \dd x. \)


\(31.\)

Show that \(\displaystyle \int \frac{\sin(x)}{x} \, \dd x = \sin(x) \ln |x| - \int \cos(x) \ln |x| \, \dd x.\)


\(32.\)

Let \(k\) be a twice-differentiable function whose second derivative is positive for all \(x.\) It is known that \(k(0) = 0, k(2) = 3,\) and \(k'(2) = 4.\) Show that \(\displaystyle 0 \lt \int_0^2 x \sin(x) k''(x) \, \dd x \leq 5.\)


\(33.\)

Consider the family of functions \(f(x) = x^n e^x,\) where \(n\) is a positive integer. Using integration by parts, \(\displaystyle \int f(x) \, \dd x\) can be rewritten as a series of functions of \(x.\) Find this series.


\(34.\)

Consider the family of integrals \(\displaystyle \int x^p e^x \, \dd x,\) where \(p\) is a positive constant. Show that integration by parts successfully yields an antiderivative for \(p = 1, 2, 3, \dots\) but not for \(\displaystyle p = \frac{1}{2}, \frac{3}{2}, \frac{5}{2}, \dots.\)


\(35.\)

The density of sprinkles on a circular cake as a function of the distance, \(r,\) from the center of the cake is given by \(\displaystyle s(r) = \frac{1}{10}e^{r/3},\) where \(r\) is measured in inches and \(s(r)\) is measured in sprinkles per square inch. The total number of sprinkles on the cake is given by the expression \(\displaystyle \int_0^{10} 2 \pi r s(r) \, \dd r.\) Find this value.


\(36.\)

The rate at which the number of followers of a Twitter user changes as a function of time, \(t,\) is given by \(\ell(t) = 10 - te^{-t/40},\) where \(t\) is measured in hours and \(\ell(t)\) is measured in followers per hour. The user initially has \(30\) followers. Find how many followers the user will have after \(6\) hours.


\(37.\)

The velocity of a car traveling on a straight road is given by \(\displaystyle v(t) = 30 - t^2 e^{-t/6},\) where \(t\) is measured in seconds and \(v(t)\) is measured in meters per second. The car is originally located \(400\) meters from Town \(A.\) Derive an expression for the distance of the car from Town \(A\) as a function of \(t.\)


\(38.\)

Using the substitution \(k = \ln(x),\) the integral \(\displaystyle \int \ln(x) \, \dd x\) can be re-expressed as \(\displaystyle \int k e^k \, \dd k.\) Show this process and, using integration by parts, verify that the result is the expected expression \(x \ln(x) - x + C.\)


\(39.\)

Show that \(\displaystyle \int \cos^n(\theta) \, \dd \theta = \frac{1}{n} \sin(\theta) \cos^{n - 1}(\theta) + \frac{n - 1}{n} \int \cos^{n - 2}(\theta) \, \dd \theta. \)
