10.10 — Absolute and Conditional Convergence

In this section we will discuss absolute and conditional convergence, which are descriptions of the types of convergence for series. These descriptors are important, as we will see in further sections, for determining how "strong" a particular type of convergence is. Absolute convergence describes a series whose absolute value converges, while conditional convergence describes a series that converges but not absolutely (its absolute value diverges.)

If \(\sum|a_n|\) converges, then \(\sum a_n\) is absolutely convergent.

If \(\sum a_n\) converges but \(\sum |a_n|\) diverges, then \(\sum a_n\) is conditionally convergent.

Determining whether a series is absolutely convergent can be another convergence test for series. If the convergence of \(\sum a_n\) is unknown, but it is known that \(\sum |a_n|\) converges, then it can be said that \(\sum a_n\) converges, too. We will showcase a proof of this statement at the end of this section.

Theorem About Absolute Convergence

If \(\sum|a_n|\) converges, then so does \(\sum a_n\).

Example 1

Determine if the following series converges conditionally, converges absolutely, or diverges. \[\sum_{n = 3}^{\infty} a_n = \sum_{n = 3}^{\infty} \frac{(-1)^n}{\sqrt{n}}.\]

Notice that the terms of {\(a_n\)} are alternating and decrease in magnitude to zero. Furthermore, \(\displaystyle \lim_{n \to \infty} \left(\frac{1}{\sqrt{n}}\right)^n = 0\), so we conclude that \(\sum a_n\) converges by the alternating series test.

However, now let us consider \(\sum |a_n|\), which is \(\sum \frac{1}{\sqrt{n}}\) in this case. This series is a p-series with \( p = 0.5 \lt 1\) and therefore diverges. Therefore, we know that \(\sum a_n\) converges but that \(\sum |a_n|\) diverges, so we conclude that the series \(\displaystyle \sum_{n = 3}^{\infty} \frac{(-1)^n}{\sqrt{n}}\) converges conditionally.

Example 2

Determine whether the series \( \displaystyle \sum_{n = 1}^{\infty} \frac{\sin(n)}{n^2}\) converges or diverges.

Let us first consider the series \(\displaystyle \sum_{n = 1}^{\infty} \left| \frac{\sin(n)}{n^2} \right| \). We can determine easily the convergence of this series by performing a direct comparison with \(\displaystyle \sum_{n = 1}^{\infty} \frac{1}{n^2} \). $$ 0 \leq \sum_{n = 1}^{\infty} \left| \frac{\sin(n)}{n^2} \right| \leq \underbrace{\sum_{n = 1}^{\infty} \frac{1}{n^2}}_{\textrm{converges}}. $$ Thus, \( \displaystyle \sum_{n = 1}^{\infty} \left| \frac{\sin(n)}{n^2} \right|\) converges by direct comparison.

We therefore note that, because \(\displaystyle \sum_{n = 1}^{\infty} \left| \frac{\sin(n)}{n^2} \right|\) converges, so does \( \displaystyle \sum_{n = 1}^{\infty} \frac{\sin(n)}{n^2}\) by the definition of absolute convergence.

Example 3

We can determine easily the convergence of this series by performing a direct comparison with \(\displaystyle \sum_{n = 1}^{\infty} \frac{1}{n^2} \). $$ 0 \leq \sum_{n = 1}^{\infty} \left| \frac{\sin(n)}{n^2} \right| \leq \underbrace{\sum_{n = 1}^{\infty} \frac{1}{n^2}}_{\textrm{converges}}. $$ Thus, \( \displaystyle \sum_{n = 1}^{\infty} \left| \frac{\sin(n)}{n^2} \right|\) converges by direct comparison.

We therefore note that, because \(\displaystyle \sum_{n = 1}^{\infty} \left| \frac{\sin(n)}{n^2} \right|\) converges, so does \( \displaystyle \sum_{n = 1}^{\infty} \frac{\sin(n)}{n^2}\) by the definition of absolute convergence.
