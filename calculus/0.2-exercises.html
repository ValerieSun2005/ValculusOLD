0.2 Exercises (Review of Quadratics)

These problems are meant to be done WITHOUT a calculator.

(1) Find the values of \(x\) for which \(x^2 + 6x + 9 = 0\).

(2) Find the values of \(x\) for which \(3x^2 + 7x = 13\).

(3) Find the values of \(x\) for which \(-5x^2 + 12x = 2x - 4\).

(4) How many real zeros does the function \(x^2 + x - 4\) have?

(5) How many real zeros does the function \(-2x^2 + 16x - 5\) have?

(6) Let \(f(x) = 2x^2 + 5x + c\). Find the value of \(c\) for which \(f\) has one real zero.

(7) Let \(g(x) = x^2 + bx + 2\). Find the interval of the values of \(b\) for which \(g\) has no real zeros.

(8) The function \(x^2 - 10x + 8\) will be rewritten in the form \((x-p)^2 + q\). Find \(p + q\).

(9) Find the minimum point of the function \(h(x) = x^2 + 10x - 4.\)

(10) Let \(f(x) = -x^2 - 6x + 7\). Find the \(x\)-coordinate of the maximum point of \(f\).

(11) Let \(h(x) = x^2 + 4x - 6\). Which of the following statements is/are true?

I. \(h\) has two real zeros.

II. The solutions to \(h\) are \(-4 \pm \sqrt{10}\).

III. The coordinates of the maximum point of \(h\) are \((-2,10)\).
