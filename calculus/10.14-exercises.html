10.14 Exercises (The Lagrange Error Bound)

A calculator is required for these problems.

(1) Use a second-degree Taylor series for \(e^x\) to approximate \(e^{4}\). Find the error in the approximation.

(2) Approximate \(\cos(0.77)\) using the sixth-degree Maclaurin series for \(\cos(x)\), given by \(1 - \frac{x^2}{2!} + \frac{x^4}{4!} - \frac{x^6}{6!}\). What is the maximum possible value for the difference between \(\cos(0.9)\) and your Maclaurin approximation?

(3) Use a ninth-degree Maclaurin series to approximate \(\sin(1.3)\). Find an interval in which the true value of \(\sin(1.3)\) must reside.

(4) Use a fourth-degree Maclaurin series to approximate \(\cos(0.8)\). Letting \(M = 1\), find the interval in which the true value of \(\cos(0.8)\) must reside.

(5) John believes that \(0.477\) is a plausible value for \(\cos(0.8)\). Using the interval found in Question (4), explain why John's value is not a possible value for \(\cos(0.8)\).

(6) A function \(g\) has derivatives of all orders, with \(g(2) = 3, g\,'(2) = -1, g\,''(2) = 6\), and \(g^{(3)}(2) = -14\). It is known that \(|g^{(4)}(x)| \leq 23 \) for all \(x \in [2,4]\). Find the maximum error produced in approximating \(g(2.2)\) with a Taylor series centered at \(x = 2\).

(7) The function \(v\) has derivatives of all orders, with \(v(4) = 2, v \, '(4) = -3, v \, ''(4) = -9,\) and \(v^{(3)}(4) = 5\). It is known that \(|v^{(4)}(4)| \leq 420.69 \) for all \(x\). Find the maximum error produced in approximating \(v(5.2)\) with a Taylor series centered at \(x = 4\).

(8) Let \(f\) be a function with derivatives of all orders. The \(n\)th derivative of \(f(x)\) is given by the general form \(f^{(n)}(x) = xn^2 + 8n - 7 \) for \(n \geq 0\). Let \(T_{3}(x)\) denote the third-degree Taylor series of \(f\) centered at \(x = 2\). Find the interval in which \(|f(3.8)-T_{3}(3.8)|\) must reside.

(9) A calculator uses Maclaurin series to evaluate sinusoidal functions. The screen can display up to ten digits. Find the minimum number of terms necessary to approximate \(\sin(0.42)\) for all ten digits to be accurate.

(10) The function \(\sin{(x)}\) is approximated using its seventh-degree Maclaurin series. Find all the values \(x\) for which the error of the approximation at most \(0.0001\).

(11) Show that the Maclaurin series for \(e^x\) converges to \(e^x\) for all \(x\).

(12) Show that the Maclaurin series for \(\sin(x)\) converges to \(\sin(x)\) for all \(x\).

(13) Find the values of \(x\) for which the fourth-degree Maclaurin series of \(\cos(x)\) has an error bound greater than \(1\).

(14) A power series for \(\tan^{-1}{(x)}\) is given by \(\sum_{n = 0}^{\infty} (-1)^n \frac{x^{2n+1}}{2n+1}\). Show that the series diverges from \(\tan^{-1}(x)\) for \(|x| > 1\).

Problems 15–18 are multiple-choice questions. Select the best answer from the choices provided.

(15) The Lagrange Error Bound . . .

(A) provides the exact value of the error of a Taylor approximation.

(B) is negative when \((x - c) < 0\).

(C) is numerically less than the first omitted term of the Taylor approximation.

(D) produces a value greater than or equal to the true error of a Taylor approximation.

(E) Answers (C) and (D) are correct.

(16) A Taylor series centered at \(x = 2\) is used to approximate \(f(x)\). Which of the following is FALSE?

(A) The true value of \(f(x)\) is bounded between \((\text{approximation} - \text{error})\) and \((\text{approximation} + \text{error})\).

(B) The error bound will decrease if a higher degree Taylor series is used.

(C) The error bound will increase if \(x\) is closer to \(2\).

(D) The error bound can never be negative.

(E) All of the above statements are true.

(17) The fifth-degree Maclaurin series for \(\sin(x)\) is used to approximate \(\sin(9.8)\). Which of the following options can be used to decrease the value of the error of the approximation?

(A) Use the same series to approximate \(\sin(-2.766)\) instead.

(B) Decrease the degree of the series to \(3\).

(C) Increase the value of \(M\).

(D) Use the Alternating Series Error Bound to bound the error as \(|\frac{9.8^7}{7!}| \).

(E) None of the above options will decrease the error.

(18) The Lagrange Error Bound is used to bound the error, denoted \(R\), of a Taylor approximation of \(f(0.2)\), denoted \(T\). It is known that the Taylor series used converges to \(f(x)\) for all \(x\) and is centered at \(x = 0.4\). Which of the following options is TRUE about the interval \([T - R, T + R]\)?

(A) In very extreme cases, \(f(0.2)\) may fall outside of the interval.

(B) The range of the interval will decrease if the Taylor series is centered at \(x = -0.2\) instead.

(C) \(f(0.2)\) will be closer to \(T + R\) because \(x > 0\).

(D) \(f(0.2)\) will be closer to \(T - R\) because \(x > 0\).

(E) The range of the interval will decrease with higher-degree Taylor approximations.

Problems 19–22 are conceptual questions that require you to explain your reasoning.

(19) Explain what may happen to the error bound if \(M\) is not let to be greater than \(f^{(n+1)}(z)\) for all \(z \in [x,c]\).

(20) It is of course possible to let \(M\) equal some arbitrarily large number. Explain why this initialization is not practical.

(21) Explain why it is logical that the error bound varies directly with \(|x - c|\) and inversely with \(n\).

(22) What are the differences between the Lagrange Error Bound and the Alternating Series Error Bound?
