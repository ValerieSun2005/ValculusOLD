2.2 Exercises — Finding Derivatives

For exercises 1–7, evaluate the following derivatives.

\(1.\)
\(\displaystyle \frac{\textrm{d}}{\textrm{d} x} (x^3) \)

\(2.\)
\(\displaystyle \frac{\textrm{d}}{\textrm{d} x} (5x^8) \)

\(3.\)
\(\displaystyle \frac{\textrm{d}}{\textrm{d} x} (9x^4 + 8x^2 - 15x -9e) \)

\(4.\)
\(\displaystyle \frac{\textrm{d}}{\textrm{d} x} (\pi^2) \)

\(5.\)
\(\displaystyle \frac{\textrm{d}}{\textrm{d} x} (10\sqrt{x} - 20 \sqrt[3]{x}) \)

\(6.\)
\(\displaystyle \frac{\textrm{d}}{\textrm{d} x} (2x^2 - 8x + 9) \Big|_{x = 2} \)

\(7.\)
\(\displaystyle \frac{\textrm{d}}{\textrm{d} x} (\sin(x) - e^x) \Big|_{x = 0} \)

\(8.\)
Write the equation of the tangent line to \(y = x^2 - 4x + 5\) at \(x = 4\).

\(9.\)
Write the equation of the tangent line to \(y = \sin(x)\) at \(\displaystyle x = \frac{\pi}{2} \).

\(10.\)
Write the equation of the tangent line to \(y = e^x\) at \(x = \ln(3)\).

A calculator is required for these problems.

(11) Approximate the value of \(\sin(1.3)\) using a tangent line approximation to \(y = \sin(x)\) at \(\displaystyle x = \frac{\pi}{2} \).

(12) Approximate the value of \(\cos(3.8)\) using a tangent line approximation to \(y = \cos(x)\) at \(x = \pi \).

(13) Find the interval of values \(x\) for which the tangent line approximation of \(\sqrt{x}\) centered at \(x = 25\) has an error of less than \(5\) percent.

(14) Show that any given tangent line approximation of \(\sin(x)\) and \(\cos(x)\) has a maximum error of \(200\) percent.

(15) Show that \(\displaystyle \frac{\textrm{d}}{\textrm{d}x} \Big(f(x)g(x)\Big) \ne \frac{\textrm{d}}{\textrm{d}x}(f(x)) \cdot \frac{\textrm{d}}{\textrm{d}x}(g(x)) \). (Hint: Try some polynomial expressions.)
