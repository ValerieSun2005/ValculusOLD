3.1 Exercises Solutions — Mean Value Theorem and Rolle's Theorem

QUESTION 1
Find the values that satisfy the Mean Value Theorem for \(f(x) = \sin(2x)\) over \(0 \leq x \leq \frac{\pi}{2}.\)
SOLUTION The average rate of change of \(f(x)\) over \(0 \leq x \leq \frac{\pi}{2}\) is \[\frac{\sin \frac{\pi}{2} - \sin 0}{\frac{\pi}{2} - 0} = 0 \per \] Because \(f(x)\) is continuous on \(0 \leq x \leq \frac{\pi}{2}\) and differentiable on \(0 \lt x \lt \frac{\pi}{2},\) the Mean Value Theorem (or Rolle's Theorem, given that the average rate of change is zero) guarantees a value \(c\) such that \[f'(c) = 0 \per\] We then have \[f'(c) = 2 \cos(2c) = 0 \implies \boxed{c = \frac{\pi}{4}}\] as the value that satisfies the Mean Value Theorem for \(f\) over \([0, \frac{\pi}{2}].\)
QUESTION 2
Find the values that satisfy the Mean Value Theorem for \(f(x) = 3x^2 - 8x + 11\) over \(1 \leq x \leq 4.\)
SOLUTION The average rate of change of \(f(x)\) over the interval \(1 \leq x \leq 4\) is \[\frac{f(4) - f(1)}{4 - 1} = 7 \per \] \(f(x)\) is a polynomial, so it's continuous over \(1 \leq x \leq 4\) and differentiable over \(1 \lt x \lt 4.\) Thus, the Mean Value Theorem guarantees a value \(c\) in \((1, 4)\) such that \[f'(c) = 7 \per\] Solving, we have \[ f'(c) = 6c - 8 = 7 \implies \boxed{c = \frac{5}{2} \per} \]
QUESTION 3
Find the values that satisfy the Mean Value Theorem for \(f(x) = \frac{1}{3} x^3 - x^2 -3x - 15\) over \(0 \leq x \leq 3.\)
SOLUTION \(f(x)\) is a polynomial, so it's continuous for \(0 \leq x \leq 3\) and differentiable for \(0 \lt x \lt 3.\) Therefore, the Mean Value Theorem guarantees a value \(c\) in \((0, 3)\) such that \[f'(c) = \frac{f(3) - f(0)}{3 - 0} = -3 \per\] Solving, we have \[ \ba f'(c) = c^2 - 2c - 3 &= -3 \\ c^2 - 2c &= 0 \implies c = 0 \comm c = 2 \per \ea \] \(\boxed{c = 2}\) is in \((0, 3),\) so this value satisfies the Mean Value Theorem for \(f\) over \([0, 3].\)
QUESTION 4
Find the values that satisfy the Mean Value Theorem for \(f(x) = |2 - x|\) over \(0 \leq x \leq 4.\)
SOLUTION \(f(x)\) is not differentiable at \(x = 2,\) so no value satisfies the Mean Value Theorem over \(0 \leq x \leq 4.\)
QUESTION 5
Find the values that satisfy the Mean Value Theorem for \(f(x) = 5e^x\) over \(0 \leq x \leq \ln 3.\)
SOLUTION Exponential functions are continuous and differentiable for all values in their domain. Thus, \(f(x)\) is continuous for \(0 \leq x \leq \ln 3\) and differentiable for \(0 \lt x \lt \ln 3.\) The Mean Value Theorem then guarantees a value \(c\) in \((0, \ln 3)\) such that \[f'(c) = \frac{f(\ln 3) - f(0)}{\ln(3) - 0} = \frac{10}{\ln 3} \per\] Solving, we have \[ \ba f'(c) = 5e^c &= \frac{10}{\ln 3} \\[2ex] e^c &= \frac{2}{\ln 3} \\[2ex] c &= \boxed{\ln \left( \frac{2}{\ln 3} \right) \per} \ea \]
QUESTION 6
Find the values that satisfy the Mean Value Theorem for \(f(x) = \frac{1}{3} x^3 - x^2 -x + 8\) over \(-3 \leq x \leq 3.\)
SOLUTION \(f\) is continuous on \([-3, 3]\) and differentiable on \((-3, 3).\) Thus, the Mean Value Theorem guarantees a value \(c\) in \((-3, 3)\) such that \[f'(c) = \frac{f(3) - f(-3)}{3 - (-3)} = 2 \per\] Solving, we have \[ \ba f'(c) = c^2 - 2c - 1 &= 2 \\ c^2 - 2c - 3 &= 0 \com \\ \ea \] whose solution in \((-3, 3)\) is \(\boxed{c = -1 \per}\)
QUESTION 7
Consider the function \(g(x) = ax^2 + bx,\) where \(a\) and \(b\) are constants. Show that \(c = k/2\) satisfies the Mean Value Theorem for \(g\) over \(0 \leq x \leq k.\)
SOLUTION Because \(g(x)\) is a polynomial, it's differentiable for all \(x.\) Moreover, the average rate of change of \(g\) over \([0, k]\) is \[ \ba \frac{g(k) - g(0)}{k - 0} &= \frac{ak^2 + bk}{k} \\[2ex] &= ak + b \per \ea \] Therefore, the Mean Value Theorem guarantees a value \(c\) in \((0, k)\) such that \[g'(c) = ak + b \per \] Solving, we have \[ \ba g'(c) = 2ac + b &= ak + b \\ c &= k/2 \per \ea \]
QUESTION 8
Consider Figure 7, in which the graph of \(y = \) \(f(x)\) is shown. For \(-2 \lt x \lt 3,\) \[f(x) = (x - 1)^2 - 6 \per\]
  1. Find the average rate of change of \(f(x)\) over \(-2 \leq x \leq 3.\)
  2. Determine the value \(c\) that satisfies the Mean Value Theorem for \(f\) over \([-2, 3].\) State the conditions of the Mean Value Theorem.
  3. The average rate of change of \(f(x)\) over \(-6 \leq x \leq 3\) is \(0.\) Does \(f\) satisfy Rolle's Theorem over this interval?
SOLUTION
  1. The average rate of change of \(f(x)\) over \(-2 \leq x \leq 3\) is \[\frac{f(3) - f(-2)}{3 - (-2)} = -1 \per\]
  2. \(f(x)\) is continuous on \(-2 \leq x \leq 3\) and differentiable on \(-2 \lt x \lt 3.\) Thus, the Mean Value Theorem guarantees a value \(c\) in \((-2, 3)\) such that \(f'(c) = -1.\) We then get \[f'(c) = 2(c - 1) = -1 \implies \boxed{c = \frac{1}{2} \per} \]
  3. No, because \(f(x)\) is not differentiable at every point in \(-6 \lt x \lt 3.\) [At \(x = -2,\) there is a sharp turn in the graph of \(y = f(x).\)] Therefore, the conditions of Rolle's Theorem are not met.
QUESTION 9
The amount of snow, in inches, on a road after \(t\) hours is modeled by \[s(t) = 5 \left(1 - e^{-t/5}\right) \per\] For what value of \(k,\) \(0 \leq k \leq 5,\) is \(s'(k)\) equal to the average rate of change of \(s(t)\) over \(0 \leq t \leq 5 \ques\) (Calculator required.)
SOLUTION We find the average rate of change of \(s(t)\) over \(0 \leq t \leq 5\) to be \[\frac{s(5) - s(0)}{5 - 0} \approx 0.632 \; \text{inches}/\text{hr} \per\] We then have \(s(t) = 0.632\) at \(\boxed{t \approx 2.293 \per}\)
QUESTION 10
Let \(f\) be the polynomial defined by \(f(x) = x^5 + 6x^3 + 4x + 2.\) Note that \(f(-1) = -9\) and \(f(0) = 2.\) Using Rolle's Theorem, show that \(f(x) = 0\) has exactly one real solution.
SOLUTION \(f(x)\) is a polynomial, so it is continuous for all \(x.\) Therefore, because \(f(-1) = -9\) and \(f(0) = 2,\) the Intermediate Value Theorem guarantees a solution to \(f(x) = 0\) within \(-1 \leq x \leq 2.\) We now show that there are no other zeros of \(f\) using a Proof By Contradiction. We first start by assuming that there are two zeros of \(f \col\) \(x = a\) and \(x = b.\) It then follows that \[f(a) = f(b) = 0 \per\] Because \(f\) is a polynomial, it is continuous on \([a, b]\) and differentiable on \((a, b).\) Thus, Rolle's Theorem states that there must be a value \(c\) in \((a, b)\) such that \(f'(c) = 0.\) Differentiating, we attain \[5c^4 + 18c^2 + 4 = 0 \com\] which has no solution. Therefore, we have a contradiction: The only condition of Rolle's Theorem that \(f\) does not satisfy is \(f(a) = f(b) = 0.\) We conclude, therefore, that there must only be one zero of \(f.\)
