5.5 Exercises — Problem 14

\((14)\) Determine whether \(\displaystyle \int_{0}^e \ln(x + 1) \, \textrm{d} x \) is a proper or improper integral. If it is improper, classify the type (i.e, Type I or Type II).

\(\displaystyle \ln(x + 1)\) is continuous for all \(x \gt -1\). Therefore, the interval over which we integrate, \([0,e]\), does not contain any discontinuities. This implies that the integral is proper.
