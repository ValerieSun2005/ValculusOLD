5.5 Exercises — Problem 26

\((26)\) It is known that \(\displaystyle 0 \lt \frac{1}{x^{5/9}} \leq g(x)\), where \(g\) is continuous for all \(x\). Show that \(\displaystyle \int_1^\infty g(x) \, \textrm{d} x \) is divergent.

Since \(\displaystyle 0 \lt \frac{1}{x^{5/9}} \leq g(x)\) and \(g\) is continuous for all \(x \geq 1\), it follows that $$ 0 \lt \int_1^\infty \frac{1}{x^{5/9}} \, \textrm{d} x \leq \int_1^\infty g(x) \, \textrm{d} x. $$ We notice that \(\displaystyle \int_1^\infty \frac{1}{x^{5/9}} \, \textrm{d} x\) diverges because it is a \(p\)-integral with \(\displaystyle p = \frac{5}{9}\). Therefore, using the comparison test, \(\displaystyle \int_1^\infty g(x) \, \textrm{d} x\) diverges too. This concept can intuitively be considered as the fact that a number "larger than infinity" will "still be infinity."
