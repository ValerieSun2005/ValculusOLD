5.5 Exercises — Problem 23

\((23)\) Find all the values of \(n\) for which \(\displaystyle \int_{1}^{\infty} \frac{1}{x^{5n - 1}} \, \textrm{d} x \, \) converges.

Note that this integral is analogous to the form \(\displaystyle \int_1^\infty \frac{1}{x^p} \, \textrm{d} x \). We understand that the \(p\)-integral will converge if \(p \gt 1\). Therefore, we set up the inequality $$ 5n - 1 \gt 1, $$ and we see that $$ \bbox[border: 2px solid white, 2pt]{n \gt \frac{2}{5}.}$$
