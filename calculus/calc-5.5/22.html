5.5 Exercises — Problem 22

\((22)\) Find all the values of \(n\) for which \(\displaystyle \int_{2}^{\infty} \frac{3}{x^{1 - 3n}} \, \textrm{d} x \, \) converges.

We can rewrite the integral as $$ 3 \int_2^\infty \frac{1}{x^{1- 3n}} \, \textrm{d} x, $$ which is analogous to the \(p\)-integral \(\displaystyle \int_2^\infty \frac{1}{x^p} \, \textrm{d} x \). We understand that the \(p\)-integral will converge if \(p \gt 1\). Therefore, we set up the inequality $$ 1 - 3n \gt 1, $$ in which we can solve for \(n\). Solving, we see that $$ \bbox[border: 2px solid white, 2pt]{n \lt 0.}$$
