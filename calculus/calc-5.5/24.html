5.5 Exercises — Problem 24

\((24)\) Find all the values of \(n\) for which \(\displaystyle \int_{1}^{\infty} \frac{1}{x^{n^2 - 4n + 4}} \, \textrm{d} x \, \) diverges.

We observe that this integral is in the form \(\displaystyle \int_2^\infty \frac{1}{x^p} \, \textrm{d} x \), where \(p = n^2 - 4n + 4\). In this case, the integral will diverge if \(p \lt 1\). Therefore, we set up the inequality $$ n^2 - 4n + 4 \lt 1, $$ $$ n^2 - 4n + 3 \lt 0 $$ $$ (n - 3)(n - 1) \lt 0, $$ from which we obtain two possibilities: either \(1 \lt n \lt 3\) OR \(n \lt 1, n \gt 3\). We can test the inequality by letting \(n = 0\). Doing so gives $$ (0 - 3)(0 - 1) \lt 0 $$ $$ 3 \lt 0, $$ which is FALSE. Therefore, \(1 \lt n \lt 3\) is the correct solution set, while \(n \lt 1, n \gt 3\) is extraneous. $$ \bbox[border: 2px solid white, 2pt]{1 \lt n \lt 3.}$$
