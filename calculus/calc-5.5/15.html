5.5 Exercises — Problem 15

\((15)\) Determine whether \(\displaystyle \int_{-3}^{-2} \frac{1}{x^3 + 1} \, \textrm{d} x \) is a proper or improper integral. If it is improper, classify the type (i.e, Type I or Type II).

Notice that \(\displaystyle \frac{1}{x^3 + 1}\) is continuous for all \(x \ne -1\). Because the interval over which we integrate does not contain \(x = -1\), the integral is proper.
